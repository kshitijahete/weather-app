Description:
Developed a responsive Weather Forecast Web Application that provides real-time weather information for any city. The application fetches live data from the OpenWeatherMap API and displays temperature, weather conditions, humidity, wind speed, sunrise, sunset, and local time. The project focuses on clean UI design, API integration, asynchronous JavaScript, and user-friendly error handling.

Technologies Used:
HTML5, CSS3, JavaScript (ES6), OpenWeatherMap API

Key Features:

Search weather details by city name

Displays real-time temperature, weather description, humidity, and wind speed

Shows sunrise, sunset, and local time using timezone data

Responsive and modern UI design

Error handling for invalid city names and network issues
